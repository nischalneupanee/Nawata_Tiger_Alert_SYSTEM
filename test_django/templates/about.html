Know Live With Paws

Live with Paws is an innovative Tiger-Human Conflict Mitigation System designed to ensure the peaceful coexistence of humans and tigers in Nepalâ€™s tiger-prone areas. Using advanced machine learning techniques we can detect tiger movements in real-time, displaying tiger activity on an interactive map.

Our goal is to create a sustainable environment that mitigates tiger-human conflicts. The project incorporates light and sound alert systems with devices like Arduino for areas where tiger activity is detected, ensuring safety for both humans and wildlife.

This project, designed for a hackathon, promotes wildlife conservation using modern technology, creating a safer and thriving environment for both tigers and humans.
